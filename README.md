# bg-frontend

Устанавливаем зависимости
```bash
pnpm i
```
Создаем .env файл в корне проекта по подобию .env.example

Для запуска dev используем
```bash
npm run dev
```
Для запуска storybook
```bash
npm run dev:storybook
```

Так же доступы команды для проветки lint 
```bash
npm run lint
```

и типов typescript
```bash
npm run typecheck
```
